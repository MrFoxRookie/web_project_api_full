# Tripleten web_project_around_express

A continuación, se tiene un proyecto back-end utilizando el framework Express.js para Node.js. Se puede observar un archivo app.js el cual funciona como el punto de entrada del servidor. Dentro de este archivo, se crea una instancia de la aplicación app, la cual permite el procesamiento de solicitudes de los usuarios a rutas específicas mediante el uso de 2 routers, los cuales están destinados para el manejo de la información de usuarios y tarjetas.

Dicho lo anterior, para el manejo de las distintas solicitudes, se empleó la utilización de controladores conectados a sus modelos correspondientes, permitiendo la validación y verificación de los datos enviados por los usuarios al servidor. Además, tales controladores incluyen la lógica encargada del procesamiento de la base de datos del proyecto, así como de la devolución de errores en caso de ser necesario.
